# voting-client
React and Redux Client for the voting-server 
http://teropa.info/blog/2015/09/10/full-stack-redux-tutorial.html

## Setup
```
git clone https://github.com/jsfeb26/voting-client.git
cd voting-client
npm install
```

## Running Tests
To run all tests:
```
npm run test
```
Or to watch tests:
```
npm run test:watch
```
